# Personal Portfolio Website

Just a generic-ish personal portfolio website showcasing my projects, technical skills, research, professional experience as well as some of my writing.

## Overview
This project serves as a central place to present:
- About Me
- Technical Skills
- Selected Projects
- Research Work
- Resume/CV
- Contact Information

## Features
- Responsive design for desktop and mobile
- Project showcase with descriptions and links
- Resume download
- Contact section
- Clean and simple navigation

## Tech Stack
Built with:
- Astro
- TypeScript
- HTML5
- CSS3
- JavaScript
- Node.js
- npm
